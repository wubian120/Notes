# Github Note


## 基本概念

- 工作区
本地电脑上的工作目录。
- 暂存区
就是称为stage（或者叫index）的暂存区，还有Git为我们自动创建的第一个分支master，以及指向master的一个指针叫HEAD。
- 版本库 Repository 工作区隐藏目录.git
